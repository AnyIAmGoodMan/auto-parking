# Automatic Parking System

Статус: Работающий прототип.

Проект представляет собой автоматизированную систему парковки для шести машин. При подъезде к парковке машинка показывает QR-код, считыватель распознаёт его, после чего парковка автоматически поворачивается к указанному для этой машины месту. Машинка въезжает на парковку. Система рассчитана на шесть машино-мест.

## Состав проекта

Проект включает три программных модуля и механическую часть:

- Код для считывателя QR-кода.
- Код для управления парковкой.
- Код для машинки с управлением от пульта.

Также в проекте присутствуют чертежи и фотографии деталей, изготовленных на фрезеровальном станке из фанеры.

## Принцип работы

1. Машинка подъезжает к парковке.
2. Считыватель QR-кода распознаёт код, привязанный к конкретной машине.
3. Парковка поворачивается к нужному месту.
4. Машинка въезжает на парковку.
5. Система фиксирует занятость места.
6. В системе предусмотрено шесть мест для машин.

## Структура репозитория

- /Код - коды для всех элементов
- /Чертежи - чертежи деталей, изготовленных из фанеры.
- /Фотографии - фотографии готовых деталей и собранной конструкции.
- /Видео - демонстрационное видео
- /3д модели - модели для всех элементов, в частности электросхемы

## Компоненты системы

- Считыватель QR-кода.
- Парковка с механизмом вращения.
- Машинка с дистанционным управлением.
- Шесть машино-мест.
- Фанерные детали, изготовленные на фрезеровальном станке.


## Сборка механической части

Все детали спроектированы для изготовления из фанеры на фрезеровальном станке. Чертежи доступны в папке /Чертежи. Фотографии готовых деталей и собранной конструкции представлены в папке /Фото.

## Примечания

- Проект представляет собой действующий прототип.
- Система рассчитана на шесть машин.
- Управление машинкой осуществляется с пульта.
- Парковка вращается для подачи нужного места.
- Считывание QR-кода инициирует процесс парковки.
